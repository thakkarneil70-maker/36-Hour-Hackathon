# 🧪 VisoiLab — Virtual AI Learning Lab

## 🚀 Overview

EduLab is an interactive virtual learning platform designed to make complex subjects like Mathematics, Chemistry, and Programming intuitive through **visual simulations** and a **global AI tutor**.

Instead of passive learning, EduLab enables students to **explore, interact, and understand concepts visually**.

---

## ✨ Key Features

### 🧠 Global AI Tutor

* Ask questions from **any section**
* Supports **Math, Chemistry, and Coding**
* Context-aware responses
* Seamless integration across the platform

### 🎮 Gamified Learning

* Interactive simulations
* Visual understanding of complex topics
* Engaging UI for better retention

### 💻 Coding & Algorithm Visualisation

* Helps understand concepts like:

  * Backtracking
  * Dynamic Programming
* Visual-first learning approach

---

* **Frontend:** HTML, CSS, JavaScript

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/virtual-lab-hackathon.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the server:

```bash
node server.js
```

4. Open in browser:

```
http://localhost:8080
```

---

## 🔐 Environment Variables

Create a `.env` file in root:

```
API_KEY=your_api_key_here
```
